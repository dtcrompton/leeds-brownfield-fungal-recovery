# Manchester Urban Mycoforestry Potential

**Geospatial analysis identifying urban green spaces with high carbon sequestration and biodiversity enhancement potential through mycorrhizal fungi establishment**

## Project Overview

This project maps carbon sequestration potential and ecological co-benefits of establishing mycorrhizal networks in Manchester's urban parks, street trees, and community green spaces.

**Research Questions:**
1. Which sites have the highest carbon sequestration potential if mycorrhizal fungi networks are established?
2. Which sites would simultaneously deliver biodiversity co-benefits (improved tree health, soil quality, fungal diversity)?

**Approach:**
- Identify and map urban green spaces (parks, street trees, community gardens)
- Assess baseline tree species and mycorrhizal associations (ectomycorrhizal vs. arbuscular)
- Model carbon sequestration potential based on tree species, soil conditions, and fungal enhancement
- Score sites for biodiversity co-benefits (tree health improvement potential, connectivity to green corridors)
- Create interactive priority map for stakeholders (councils, NGOs, community groups)

## Study Area

**Manchester, UK** — selected because:
- Well-documented urban tree inventory (Manchester City Council)
- Previous brownfield restoration project provides local context
- Strong environmental policy framework (Manchester Climate Change Action Plan)
- Manageable geographic scale for proof-of-concept

## Data Sources

- **Tree species:** OpenStreetMap, Manchester City Council tree surveys
- **Green space boundaries:** OS MasterMap, council open data
- **Soil properties:** UKCEH Countryside Survey, LandIS
- **Soil carbon:** UKCEH soil carbon maps
- **Carbon sequestration rates:** Published literature (Averill et al. 2014, Soudzilovskaia et al. 2019)
- **Fungal occurrence data:** GBIF (optional validation)

## Methodology

### Phase 1: Data Collection & Green Space Mapping
- [ ] Identify urban parks, street trees, community gardens
- [ ] Obtain tree species data from council/OSM
- [ ] Download soil carbon and property layers

### Phase 2: Tree-Fungi Species Matching
- [ ] Classify trees by mycorrhizal type (EM vs. AM)
- [ ] Match species to suitable fungal partners
- [ ] Assess current mycorrhizal colonization potential

### Phase 3: Carbon Sequestration Modelling
- [ ] Calculate baseline soil carbon stocks
- [ ] Estimate carbon sequestration potential with fungal enhancement
- [ ] Model 10-year and 20-year projections

### Phase 4: Biodiversity Co-Benefits Assessment
- [ ] Score tree health improvement potential
- [ ] Assess connectivity to existing green corridors
- [ ] Identify fungal biodiversity potential

### Phase 5: Multi-Criteria Site Prioritization
- [ ] Combine carbon + biodiversity scores
- [ ] Filter by feasibility (site access, management constraints)
- [ ] Create interactive priority map

### Phase 6: Stakeholder Outputs
- [ ] Policy brief for Manchester City Council
- [ ] Community engagement toolkit
- [ ] Interactive web map
- [ ] Portfolio write-up
